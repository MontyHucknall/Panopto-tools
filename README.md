**Panopto Tools Chrome Extension**

## Usage

A local Chrome extension for downloading Panopto content for local viewing.

You use it to save video or audio streams and extract auto generated captions for LLM summarisation.

## Features

- Save video streams
- Save audio streams
- Extract auto captions
- Export captions as plain text

## Experimental Features

- Combine video & audio streams for a complete recording file ⚠️ In progress ⚠️

## Requirements

- Google Chrome or Chromium based browser
- Access to Panopto content through your browser session

## Installation

This extension installs in Developer mode (read the code if you are unsure).

1. Download or clone this repository
2. Open Chrome and go to chrome://extensions
3. Enable Developer mode
4. Select Load unpacked
5. Choose the extension folder

The extension now appears in your browser.

## Usage

1. Open a Panopto video
2. Start playback
3. Open the extension from the toolbar
4. Choose transcripts, streams or combine audio/video streams
